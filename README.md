# 🤖 - M4Bot
Um plugin que adiciona um bot com comandos ao seu servidor (configurável) para pocketmine 2.0.0 (Mcpe 0.14 & 0.15)
Comandos: lola, broxa (ou brocha), ship, calvo, ccme, help e version
